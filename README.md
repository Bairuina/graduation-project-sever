## 技术栈
- [Node.js >= 14](https://nodejs.org/en/)
- [Egg.js](https://eggjs.org/zh-cn/intro/)
- [MySQL](https://www.mysql.com/)
 

## 项目启动
- Node.js >= 14
- 确保已安装 [MySQL](https://www.mysql.com/)

``` bash
# 克隆项目
git clone https://github.com/Bairuina/graduation-project-sever.git

# 安装依赖
yarn

# 启动项目
yarn dev

# 项目打包
yarn build
```
